01  서비스 개요

유적지마다 담긴 역사 스토리 카드를 드래그해
나만의 하루 여정을 완성하면,
이동 구간별 교통편 · 명소별 맛집 · 날씨가
실시간으로 업데이트되는 경주 전용 여행 플래너

기존 여행 앱의 한계
검색→선택→저장의 단순 반복. 역사적 맥락 없이 위치 정보만 제공되며, 순서를 바꿀 때마다 교통편을 다시 찾아야 하는 번거로움이 있다.
경주 타임트래블의 해법
이야기를 읽고 고르는 것만으로 완벽한 여정이 완성된다. 순서 배치와 동시에 모든 실용 정보가 자동으로 갱신된다.

05  실시간 맞춤 정보 시스템

카드 배치가 바뀔 때마다 0.6초 내에 자동 업데이트된다.

🚌
구간별 교통편
"경주역 → 불국사", "불국사 → 석굴암" 등 이동 구간마다 버스·택시·도보 옵션을 각각 표시
→ 매번 따로 검색하던 번거로움 제거
🍽
명소별 맛집
배치된 카드 각각의 주변 맛집을 별도로 추천 (불국사 돌솥밥, 황리단길 한옥 카페 등)
→ 어느 곳을 방문하든 근처 식당 자동 완성
📊
여정 요약
선택 명소 수, 예상 총 소요 시간, 이동 거리를 즉시 집계
→ 무리 없는 현실적인 하루 계획 수립
🌤
날씨 안내
경주 당일 기온·미세먼지·자외선 정보와 추천 준비물 표시
→ 날씨에 따른 방문지 재조합 유도
⚡
업데이트 응답 속도 안내 OpenAPI 서버 응답 속도 및 네트워크 환경에 따라 실시간 반영이 다소 지연될 수 있으며, 이 경우 "여정 확정" 버튼을 통해 일괄 조회하는 방식으로 안정적인 정보 제공이 가능합니다. 최종 제출 버전에서는 API 캐싱 전략을 적용하여 체감 응답 속도를 최소화할 예정입니다.
06  차별화 포인트
기존 여행 플래너의 한계
네이버 지도·카카오맵: 위치 검색 중심, 스토리 없음
트리플·마이리얼트립: 찜 목록 방식, 순서 변경 시 정보 미갱신
경주시 공식 앱: 단방향 안내, 사용자 커스텀 불가
경주 타임트래블의 강점
스토리 우선 — 흥미 유발 후 방문 동기 자연 생성
배치 즉시 갱신 — 교통·맛집 정보를 매번 검색할 필요 없음
경주 단일 특화 — 깊이 있는 로컬 콘텐츠 큐레이션
